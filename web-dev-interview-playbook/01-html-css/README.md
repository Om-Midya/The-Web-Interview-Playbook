# HTML & CSS Interview Preparation

HTML and CSS interviews test whether you can build **real, accessible, responsive UIs** — not whether you've memorized every CSS property.

## What Interviewers Look For

- Semantic HTML (not div soup)
- Layout skills (Flexbox, Grid — not just Bootstrap classes)
- Responsive design thinking
- Basic accessibility awareness
- CSS specificity and cascade understanding

3. `mini-tasks.md` — build 3 layouts without frameworks
4. `mock-interview.md` — practice with a friend
5. `revision-sheet.md` — day before interview

## Time Estimate

- **Minimum viable prep:** 3–4 days
- **Solid prep:** 1 week (with mini-tasks)

## Project Connection

Be ready to explain how you styled your project: layout approach, responsive breakpoints, any accessibility considerations.
