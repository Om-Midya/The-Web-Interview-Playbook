# Next.js Interview Preparation

Next.js interviews test whether you understand **rendering strategies, routing, and full-stack React** — the framework most companies use for production React apps.

## What Interviewers Look For

- SSR vs SSG vs CSR vs ISR — when and why
- App Router vs Pages Router awareness
- Data fetching patterns (server vs client)
- File-based routing and layouts
- How your Next.js project is structured

- **Minimum viable prep:** 3–5 days (if you know React)
- **Solid prep:** 1–2 weeks (with a Next.js project)

## Project Connection

Be ready to explain: which routes use SSR/SSG, why you chose App Router, how you fetch data, and your deployment setup (Vercel or other).
