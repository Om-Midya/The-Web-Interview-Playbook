# Node.js & Express Interview Preparation

Backend interviews test whether you can **build APIs, handle auth, and reason about server-side concerns** — not just copy Express boilerplate.

## What Interviewers Look For

- REST API design
- Middleware understanding
- Authentication & security basics
- Error handling and validation
- Database integration patterns
- How your project backend is structured

## Study Order

1. `topics-not-to-miss.md` — identify gaps
2. `api-design.md` + `auth-security.md` — core backend literacy
3. `interview-questions.md` — 5 questions/day
4. `project-style-questions.md` — scenario practice
5. `mock-interview.md` — practice with a friend
6. `revision-sheet.md` — day before interview

## Time Estimate

- **Minimum viable prep:** 5–7 days
- **Solid prep:** 2 weeks (with a small API project)

## Project Connection

Be ready to explain: your API routes, auth flow, database schema, error handling, and one security decision you made.
